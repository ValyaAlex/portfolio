Цистоскопия с гидробужированием мочевого пузыря

Для диагностики синдрома болезненного мочевого пузыря / интерстициального цистита при сниженной максимальной емкости мочевого пузыря по данным уродинамического исследования и дневников мочеиспускания выполняется цистоскопия с переполнением (гидробужированием) мочевого пузыря, то есть наполнением больше того объема, который пациент обычно накапливает. Только максимальное наполнение позволяет увидеть характерные для интерстициального цистита изменения слизистой мочевого пузыря в виде петехиальных кровоизлияний или язв Гуннера.

Цистоскопия с гидробужированием выполняется только по назначению врача, под наркозом. Может сопровождаться также биопсией с последующим гистологическим исследованием.
